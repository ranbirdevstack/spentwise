# 💰 SpentWise

SpentWise is a modern Salary and Expense Tracker built using **HTML, CSS, and Vanilla JavaScript**. It helps users manage their monthly income, record expenses, monitor their remaining balance, visualize spending through interactive charts, and generate downloadable PDF reports.

The application is designed with a clean and responsive dashboard, making personal finance management simple and efficient.

# 📖 About the Project

Managing personal finances manually can be difficult and time-consuming. SpentWise provides a digital solution that allows users to record their salary and expenses while automatically calculating the remaining balance.

The dashboard updates instantly whenever a new expense is added or deleted. It also provides visual analytics using Chart.js, supports currency conversion, and allows users to export their financial report as a PDF.

---

# ✨ Features

- Add monthly salary
- Read-only salary after adding
- Edit salary using Edit button
- Add unlimited expenses
- Delete expenses instantly
- Automatic balance calculation
- Real-time dashboard updates
- Expense ledger
- Currency conversion (INR / USD)
- Interactive Doughnut Chart
- PDF report generation
